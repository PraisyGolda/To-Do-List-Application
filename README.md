# To-Do List Application

## Project Description
The To-Do List Application is a web-based task management tool that helps users efficiently organize and manage their daily tasks. Users can add, edit, delete, and mark tasks as complete, with features for filtering and searching tasks. The app is designed with a responsive interface to provide a smooth experience on both desktop and mobile devices.

---

## Features
- User authentication (login/signup)  
- Add new tasks with title, description, and due date  
- Edit/update existing tasks  
- Delete tasks with confirmation  
- Mark tasks as completed  
- Filter tasks by status (Completed / Pending)  
- Search tasks by keywords  
- Responsive design for desktop and mobile  
- Real-time task updates

---

## Technologies Used
- **Frontend:** React.js, HTML5, CSS3, JavaScript  
- **Backend:** Flask / Node.js (REST API)  
- **Database:** MongoDB / SQLite  
- **Libraries/Tools:** Axios, React Router, Bootstrap/Material UI (optional)
